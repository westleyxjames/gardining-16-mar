## 🚀 How to Use

### Quick Start
```bash
cd public
python -m http.server 8000
# Open http://localhost:8000
```

### Download for Offline
1. Download entire `/public` folder
2. Open `image-downloader.html` in browser
3. Download all 6 images
4. Create `images/` folder
5. Place images in folder
6. Update HTML paths (see DOWNLOAD-GUIDE.md)
7. Test offline!

### Deploy to Web
- Netlify: Drag & drop `/public` folder
- Vercel: Connect repo and deploy
- GitHub Pages: Push to gh-pages
- Any host: Upload via FTP
